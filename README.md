# Proyecto-3---Maquina-de-Turing---Lenguajes

Manual de usuario – Proyecto #3

Al iniciar el programa, se desplegará un menú el cual pedirá elegir el lenguaje a utilizar para crear una cadena.
 <img width="775" height="356" alt="image" src="https://github.com/user-attachments/assets/0fb1c4d5-ebb8-4bda-9b97-292f3fc64d8d" />


Al ingresar cualquiera de las opciones (en este caso se usó opción 4), aparecerá un texto el cual indicará el alfabeto disponible y pedirá introducir la cadena que se quiere analizar.
 <img width="813" height="302" alt="image" src="https://github.com/user-attachments/assets/5c782aeb-80d5-4141-b67c-93412aa5da20" />


Posteriormente, al haber ingresado la cadena se desplegará otra opción que pedirá elegir entre el análisis paso a paso o por uno automático. En el caso de elegir el paso a paso, se desplegará toda la información del estado, los pasos hasta el momento y la cinta grafica. Para poder ver el siguiente paso se deberá de presionar “espacio” y posteriormente “Enter”. Al finalizar el recorrido, se desplegará la visualización del MT con la información: Cadena aceptada o rechazada, Pasos simulados y el contenido de la cadena. 
Por el otro lado, si se elige la opción 2 (Automático), el programa analizara la cadena y se saltara hasta el paso de desplegar la visualización del MT con la misma información previamente dicha.
